# Hotfix V 8.0.0 Modo Quiosco

**Implementación del Diálogo de Consentimiento en Kiosk Launcher**

A partir de esta versión, ya no será necesario que el usuario acepte manualmente los términos. Verifique la versión instalada para confirmar si la funcionalidad está disponible.

Ahora, al abrir **Kiosk Launcher** por primera vez, el usuario verá un aviso sobre el uso de datos, informando sobre la recopilación de información de las aplicaciones instaladas. Esta recopilación es necesaria para que los íconos de las aplicaciones se muestren correctamente en la pantalla de inicio cuando el dispositivo esté en **Modo Quiosco**.

El usuario tendrá dos opciones:\
✅ **Aceptar** – Permite la recopilación de información de las aplicaciones y muestra los íconos en la launcher.\
❌ **Rechazar** – Impide la recopilación de datos, mostrando solo el **Kiosk Launcher** en la interfaz.

Esta funcionalidad garantiza transparencia y control sobre los datos utilizados por la aplicación.
